1. What is Semantic HTML? 
Ans. Semantic HTML involves use of tags which breaks up the webpage into meaningful sections such as heading, navigation menu,
main content, address/contact information and footer. Use of semantic HTML helps convey meaningful information to the browsers.
_________________________________________________________________________________________________________________________________

2. What is HTML used for?
Ans. HTML is used to incorporate/embed content into a webpage such as text, video, images and audio. 
_________________________________________________________________________________________________________________________________

3. What is an attribute and where do we put it? 
Ans. Attributes provide additional information concerning the elements. Attributes are included in name/value pair format such as
name="value". Some examples of attributes are: src attribute, in case of image, which specifies location of the image. href attribute, 
in case of anchor tag, which specifies which website should be opened when the tag is clicked.
_________________________________________________________________________________________________________________________________

4. What is the h1 tag used for? How many times should I use it on a page?
Ans. h1 tag is used for main heading of the webpage such as blog name or company name. Ideally, only one h1 tag should be used per webpage.
__________________________________________________________________________________________________________________________________

5. Name two tags that have required attributes
Ans. a tag and img tag have required attributes. 
img tag requires two attributes which are src and alt. src specifies location of the image and 
alt provides a description of the image which can be read by screen readers.

a tag requires href attribute which upon clicking takes us to the destination url. 
_____________________________________________________________________________________________________________________________________

6. What do we put in the head of our HTML document? 
Ans. Elements which are usually included in the head tag are title, meta tag which provides information regarding the character section
and link tag which links to the stylesheets used for styling/designing the webpage.

7. What is an id? 
Ans. id attribute is used within the start tag for assigning a unique identification to an HTML element. 
It is used either for styling an element using CSS or manipulating an element using JavaScript. 
________________________________________________________________________________________________________________________________________ 
8. What elements can I add an id to? 
Ans. An id can be added to any HTML element however there can be only one id per webpage.

________________________________________________________________________________________________________________________________
9. How many times can I use the same id on a page? 
Ans. An id can be used/assigned only once per webpage.
_____________________________________________________________________________________________________________________________________
10. What is a class? 
Ans. A class is used to provide an identification to a HTML element or a group of related HTML elements. 
This class can thus be used by CSS for styling or by Javascript for manipulation. 
_________________________________________________________________________________________________________________________________
11. What elements can I add a class to? 
Ans. classes can be added to any elements within the body of a HTML document. 
__________________________________________________________________________________________________________________________
12. How many times can I use the same class on a page? 
Ans. There is no limit to the times for usage of the same class within a HTML document. 
____________________________________________________________________________________________________________
13. How do I get my link to open in a new tab?
Ans. Adding the attribute "target="_blank" to the anchor tag will open the link in a new tab.
________________________________________________________________________________________________________________________________________
14. What is the alt attribute in the image tag used for? 
Ans. The alt attribute in the image tag is used for providing alternate description for the image if the image cannot be displayed or in the event 
when the web user uses a screen reader. 
_____________________________________________________________________________________________________________________
15. How do I reference an id?
Ans. An id is referenced by using the hash(#) symbol immediately before the id name. 
____________________________________________________________________________________________________________________
16. What is the difference between a section and a div
Ans. section is a semantic html tag. It conveys more information since it is used to group the main content on a webpage 
whereas div does not provide any valuable information.
